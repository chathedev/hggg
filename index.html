18a. Typvärdet för antal fångade fiskar.  
Fördelning (fångade fiskar → elever): 0→8, 1→2, 2→4, 3→7, 4→4, 5→4. Flest elever (8 st)  
fångade 0 fiskar. Typvärde = 0  
18b. Medelvärdet för antalet fångade fiskar per elev.  
Totalt elever: 8 + 2 + 4 + 7 + 4 + 4 = 29. Totalt fiskar: 0·8 + 1·2 + 2·4 + 3·7 + 4·4 + 5·4 = 0 + 2 +  
8 + 21 + 16 + 20 = 67. Medel = 67 / 29 ≈ 2,3 fiskar/elev  
19. Fyrhörning med rät vinkel, en vinkel 125°, och vinklarna a och b förhåller sig 2:3. Hur stor är  
a?  
Vinkelsumman i en fyrhörning = 360°. a + b = 360 − 90 − 125 = 145°. a:b = 2:3 → a = (2/5) ·  
145 = 58°  
20a. Hur många fler elever gick i åk 7 år 2023 jämfört med år 2013?  
Avläsning av stapeldiagram: 2013 ≈ 940 elever, 2023 ≈ 980 elever (avläs noga). Åk 7 år 2013:  
30 % · 940 ≈ 282. Åk 7 år 2023: 40 % · 980 ≈ 392. Skillnad ≈ 110 fler elever (svar i intervallet  
100–115 beroende på avläsning)  
20b. Med hur många procent ökade andelen elever i åk 8 från 2013 till 2023?  
Andel åk 8: 21 % (2013) → 25 % (2023). Procentuell ökning = (25 − 21) / 21 = 4/21 ≈ 0,190 =  
ca 19 %
